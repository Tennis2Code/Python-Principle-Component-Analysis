# Housing Market PCA (2020-2024)

This project performs Principal Component Analysis (PCA) on housing market data from the years 2020 through 2024. 
 
 Outline of the PCA:
 - Loading Data 
 - Standardization
 - Visualization (scatterplot)



Used in the Project:
- Python 3.x
- pandas
- numpy
- matplotlib
- scikit-learn
